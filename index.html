Пример кода на Python


def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

Это рекурсивная функция для вычисления факториала числа.

Она проверяет, равно ли n нулю, и в этом случае возвращает 1, так как 0! = 1.

В противном случае она возвращает n, умноженное на факториал числа n-1.
